Como conectarlo

La opcion recomendada para GitHub Pages.

  1. Abre tu Google Sheet y ve a Extensiones > Apps Script.
  2. Pega el contenido de apps-script.gs.
  3. Implementa como aplicacion web, ejecutando como tu usuario.
  4. En acceso, usa "Cualquier usuario con el enlace".
  5. Pega aqui la URL terminada en /exec.
No hace falta subir el Excel a GitHub. GitHub Pages sirve la web; Google Sheets guarda los datos.
